Course 2: Data Science Essentials

Price: $15

Description: Designed to introduce students to the world of data science, this course starts with data collection and cleaning techniques. Course 3: Cybersecurity Fundamentals

Price: $20

Description: In this course, students will learn about the essential concepts and practices of cybersecurity. It covers network security, including firewalls and intrusion detection systems. Course 4: Mobile App Development (iOS and Android)

Price: $25

Description: This course teaches students how to develop mobile applications for both iOS and Android platforms. It starts with the basics of mobile app design principles and user experience. Course 5: Artificial Intelligence Introduction

Price: $30

Description: An introductory course to artificial intelligence, it covers the basic concepts and techniques of AI.
